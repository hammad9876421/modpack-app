# ModPack App

A mobile-first Minecraft Mod Browser and Modpack Builder built with React and Vite.

## Features

### Current

- React
- Vite
- Mobile-first layout
- PWA manifest
- Dark theme
- GitHub & Vercel ready

### Planned

- Modrinth search
- CurseForge search
- Mirror fallback
- Infinite scrolling
- Advanced filters
- Mod details
- One-click download
- One-click add to modpack
- ZIP export
- Download manager
- Favorites
- History
- Offline cache

## Tech Stack

- React
- Vite
- Axios
- JSZip

## Development

Start development server:

```bash
npm run dev
```

Build production version:

```bash
npm run build
```

## Git Workflow

After every successful step:

```bash
npm run build
git add .
git commit -m "Meaningful message"
git push
```
